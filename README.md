**KrakenHax** is a mobile application that manages event registrations through a waitlist-based system.  
Entrants can join event waitlists, organizers can manage participants, and administrators oversee all system operations.  
The app integrates event discovery, filtered listings, QR scanning, and automated lottery selection for fair entry.

## Summary of System Flow

1. **Entrant** browses or filters events.  
2. They **join a waitlist** for a chosen event.  
3. **Lottery system** selects winners when registration closes.  
4. **Notifications** are sent to entrants.  
5. **Organizers** manage events and participant lists.  
6. **Admins** oversee content and maintain system integrity.
